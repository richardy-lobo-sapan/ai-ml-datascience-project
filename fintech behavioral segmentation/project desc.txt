Raw transaction logs are one of the most underutilized assets in financial services. Every purchase encodes behavioral signals about lifestyle, financial discipline, and life stage. This report documents a behavioral intelligence pipeline that converts raw transaction data into user-level features, infers latent demographic and personality attributes, and segments users into actionable groups.

The pipeline addresses two goals defined in the task:
- Feature Creation: derive user-level features from transaction-level data across 7 behavioral dimensions
- User Segmentation: identify distinct user groups using both algorithmic (KMeans, GMM) and rule-based non-traditional methods

All demographic inferences in this report are probabilistic behavioral estimates derived from population-level patterns. They are not factual labels and should not be used for individual-level decisions.

Dataset is confidential
